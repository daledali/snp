Contraception Quick Start Algorithm
<-- First day of last menstrual period?
Plan --> Start method today
<-- Unprotected sex since last menstrual period?
Plan --> Start pill/patch/ring/injection today, use back-up method 1st week
<-- Urine pregnancy test result?
Plan --> Provide options counseling
Offer hormonal emergency contraception today more information
Advise that negative pregnancy test is not conclusive, but hormones will not harm embryo
Offer hormonal emergency contraception today more information
<-- Patient wants to start new method now?
Plan -->
Start pill/patch/ring/injection, use back up method 1st week
Timing: start new method TODAY even if taking emergency contraception today
If repeat urine pregnancy test in 2 weeks is negative --> continue pill/patch/ring/injection
Plan -->
Prescription for chosen method given
Patient advised to use barrier method until next menses
Start pill/patch/ring on 1st day of menses
Return for injection within 5 days of menses
